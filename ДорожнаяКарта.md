## Протокол управления

- `др` / `сводка` / `итого` — сводка всех проектов
- `проект 76` / `проект 76_in_КОРП` / `проект КОРП` — выбрать проект
- `стз N` — детализация задачи N
- `заводим задачу` — создать задачу
- `обнови статус N.M` — изменить статус этапа
- `закрываем задачу N` — завершить
- `откладываем задачу N` — отложить
- `отк отчет` — открыть последний отчёт
- `правила дорожной карты` — показать правила
