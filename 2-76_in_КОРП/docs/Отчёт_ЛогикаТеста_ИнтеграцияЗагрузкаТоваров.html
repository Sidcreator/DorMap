🔍 Детальный разбор тестов

1 ЗагрузитьСтатусыМодерацииИИдентификаторыТоваров

Назначение: Загружает статусы модерации и идентификаторы товаров из подключённых сервисов.

Параметры: Без параметров — использует настройки авторизации по умолчанию.

ВызовСервера.ЗагрузитьСтатусыМодерацииИИдентификаторыТоваров()
→ ОбщийМодульМетода("CG_Online_НоменклатураСервисов")
→ МодульМетода.ЗагрузитьСтатусыМодерацииИИдентификаторыТоваров()
2-5 ЗагрузитьКатегорииТоваров / ДеревоКатегорииТоваров / СвойстваКатегорий / ЗначенияСвойстваКатегорий

Назначение: Загрузка справочной информации о категориях товаров (плоский список, дерево, свойства, значения).

Паттерн: Все 4 теста следуют одинаковому шаблону:

ВызовСервера.{ИмяМетода}()
→ ОбщийМодульМетода("CG_Online_НоменклатураСервисов")
→ МодульМетода.{ИмяМетода}()
6 ЗагрузитьНесопоставленныеТовары С параметрами

Назначение: Загружает только товары, которые ещё не сопоставлены с номенклатурой в базе.

Ключевой параметр: ТолькоНесопоставленные = Истина

ВызовСервера.ЗагрузитьНесопоставленныеТовары()
→ ОбщийМодульМетода("CG_Online_ИнтеграцииССервисамиСервер")
→ ПараметрыСинхронизацииДанных = ПараметрыСинхронизацииДанныхПоНастройке()
→ ЗаполнитьЗначениеПараметра("ТолькоНесопоставленные", Истина)
→ ТаблицыДанных = ТаблицыДанныхВСтруктуруПараметровСинхронизацииДанных()
→ ОбщийМодульМетода("CG_Online_НоменклатураСервисов")
→ Модуль.ЗагрузитьТовары(Параметры, ТаблицыДанных)
7 ЗагрузитьТовары С пагинацией

Назначение: Основной тест загрузки товаров. Загружает страницу товаров (10 элементов) из сервисов.

Ключевые параметры:

  • НомерСтраницы = 1 — первая страница
  • КоличествоЭлементов = 10 — лимит 10 товаров (для ускорения теста)
ВызовСервера.ЗагрузитьТовары()
→ ОбщийМодульМетода("CG_Online_ИнтеграцииССервисамиСервер")
→ Параметры = ПараметрыСинхронизацииДанныхПоНастройке()
→ ЗаполнитьЗначениеПараметра("НомерСтраницы", 1)
→ ЗаполнитьЗначениеПараметра("КоличествоЭлементов", 10)
→ ТаблицыДанных = ТаблицыДанныхВСтруктуруПараметровСинхронизацииДанных()
→ ОбщийМодульМетода("CG_Online_НоменклатураСервисов")
→ Модуль.ЗагрузитьТовары(Параметры, ТаблицыДанных)
8 ЗагрузитьШтрихкоды

Назначение: Загружает штрихкоды товаров из сервисов.

ВызовСервера.ЗагрузитьШтрихкоды()
→ ОбщийМодульМетода("CG_Online_НоменклатураСервисов")
→ МодульМетода.ЗагрузитьШтрихкоды()

⚙️ Механизм резолвинга модулей

Все тесты используют паттерн «динамический резолвинг модулей» через CG_Online_UT_ОбщегоНазначения.ОбщийМодульМетода():

МодульМетода = CG_Online_UT_ОбщегоНазначения.ОбщийМодульМетода("CG_Online_НоменклатураСервисов");
Если МодульМетода <> Неопределено Тогда
    МодульМетода.ЗагрузитьТовары(...);
КонецЕсли;

Смысл: Модуль вызывается по имени. Если модуль не существует в конфигурации (например, расширение не подключено), ОбщийМодульМетода возвращает Неопределено, и тест просто пропускает вызов. Это обеспечивает совместимость с разными конфигурациями.
