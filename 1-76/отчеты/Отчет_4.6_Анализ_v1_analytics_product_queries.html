Проект: 1/76
Дата: 2026-06-25

1. Описание

Метод возвращает агрегированные данные по поисковым запросам товаров продавца. Аналогичен вкладке «Товары в поиске → Запросы моего товара» в личном кабинете Ozon.

В отличие от /v1/analytics/product-queries/details (детализация по каждому поисковому запросу), данный метод возвращает сводную аналитику по товарам (по SKU): категория, название, артикул, продажи, позиция в выдаче.

Полная аналитика доступна с подпиской Premium, Premium Plus или Premium Pro.

2. Схема запроса

POST /v1/analytics/product-queries
Поле Тип Обязательный Описание
date_from datetime ✅ Дата начала аналитики
date_to datetime ❌ Дата окончания аналитики
page int32 ❌ Номер страницы. Минимум — 0
page_size int32 ✅ Элементов на странице. Максимум — 1000
skus array[string] ✅ Список SKU (макс. 1000)
sort_by string ❌ По умолч. BY_SEARCHES. BY_SEARCHES, BY_VIEWS, BY_POSITION, BY_CONVERSION, BY_GMV
sort_dir string ❌ По умолч. DESCENDING. DESCENDING / ASCENDING
{ "date_from": "2019-08-24T14:15:22Z", "date_to": "2019-08-24T14:15:22Z", "page": 0, "page_size": 1000, "skus": ["123456789", "987654321"], "sort_by": "BY_SEARCHES", "sort_dir": "DESCENDING" }

3. Схема ответа

{ "analytics_period": { "date_from": "2019-08-24T14:15:22Z", "date_to": "2019-08-24T14:15:22Z" }, "items": [ { "category": "string", "currency": "string", "gmv": 0, "name": "string", "offer_id": "string", "position": 0, "sku": 0, "unique_search_users": 0, "unique_view_users": 0, "view_conversion": 0 } ], "page_count": 0, "total": 0 }

5. Ограничения

Условие Описание
