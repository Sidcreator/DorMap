Проект: 1/76
Дата: 2026-06-25

1. Описание

Запускает генерацию отчёта по уценённым товарам на складе Ozon. Ozon может самостоятельно уценить товар, например, при повреждении.

В результате запроса возвращается не сам отчёт, а его уникальный идентификатор (code). Чтобы получить готовый отчёт, идентификатор передаётся в метод /v1/report/info.

2. Схема запроса (create)

POST /v1/report/discounted/create {}

Тело запроса — пустой объект. Параметры не требуются.

3. Схема ответа (create)

{ "code": "REPORT_seller_products_924336_1720170405_a9ea2f27-a473-4b13-99f9-d0cfcb5b1a69" }

4. Полный маппинг полей JSON ответа (create)

# Поле JSON Тип Описание Статус
⏳ Метод не реализован (1 поле)
1 code string Уникальный идентификатор отчёта. Передаётся в /v1/report/info для получения готового отчёта ⏳

5. Схема запроса /v1/report/info

Обрабатывает идентификатор кода из метода /v1/report/discounted/create. Возвращает информацию о созданном ранее отчёте по его идентификатору.

POST /v1/report/info { "code": "REPORT_seller_products_924336_1720170405_a9ea2f27-a473-4b13-99f9-d0cfcb5b1a69" }

6. Схема ответа /v1/report/info

{ "result": { "code": "REPORT_seller_products_924336_1720170405_a9ea2f27", "status": "success", "error": "", "expires_at": "2025-11-10T11:16:00.267Z", "file": "https://cdn1.ozone.ru/s3/item-picture-6/f3/ce/f4ceae54b323213d3e61e59c323bd8e5.csv", "report_type": "seller_products", "params": {}, "created_at": "2021-11-25T14:54:55.688260Z" } }

7. Полный маппинг полей JSON ответа /v1/report/info

# Поле JSON Тип Описание Статус
⏳ Метод не реализован (8 полей)
1 result.code string Идентификатор отчёта (тот же, что передан в запросе) ⏳
2 result.status string Статус отчёта: success / failed / in_progress ⏳
3 result.error string Описание ошибки, если статус failed ⏳
4 result.expires_at datetime Дата истечения срока действия ссылки на файл ⏳
5 result.file string Ссылка на скачивание готового отчёта (CSV) ⏳
6 result.report_type string Тип отчёта (seller_products — уценённые товары) ⏳
7 result.params object Параметры отчёта (зависят от типа) ⏳
8 result.created_at datetime Дата и время создания отчёта ⏳

8. Статусы отчёта

Статус Описание
success Отчёт готов, доступен для скачивания по ссылке result.file
failed Ошибка создания отчёта, причина в result.error
in_progress Отчёт ещё генерируется, необходимо повторить запрос позже
