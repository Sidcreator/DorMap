Проект: 1/76
Дата: 2026-06-25

1. Описание

Метод возвращает данные по поисковым запросам на конкретный товар. Аналогичен просмотру данных по товару на вкладке «Товары в поиске → Запросы моего товара» в личном кабинете Ozon.

Полная аналитика доступна с подпиской Premium, Premium Plus или Premium Pro. Без подписки доступна часть показателей.

Данные за последний месяц доступны в любом интервале (кроме текущей даты — расчёт 1–2 дня). Аналитика за даты раньше месяца — только с Premium-подпиской и только по неделям.

2. Схема запроса

POST /v1/analytics/product-queries/details

Параметры заголовков

Параметр Тип Обязательный Описание
Client-Id string ✅ Идентификатор клиента
Api-Key string ✅ API-ключ

Параметры тела запроса

Поле Тип Обязательный Описание
date_from datetime ✅ Дата начала формирования аналитики
date_to datetime ❌ Дата окончания формирования аналитики
page int32 ❌ Номер страницы. Минимум — 0
page_size int32 ✅ Элементов на странице. Максимум — 1000
skus array[string] ✅ Список SKU (макс. 1000)
sort_by string ❌ По умолч. BY_SEARCHES. Доступны: BY_SEARCHES, BY_VIEWS, BY_POSITION, BY_CONVERSION, BY_GMV
sort_dir string ❌ По умолч. DESCENDING. Направление: DESCENDING, ASCENDING
{ "date_from": "2019-08-24T14:15:22Z", "date_to": "2019-08-24T14:15:22Z", "page": 0, "page_size": 1000, "skus": ["string"], "sort_by": "BY_SEARCHES", "sort_dir": "DESCENDING" }
