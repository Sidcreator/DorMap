# Система дорожных карт

Управление задачами проектов УОТ через AI-ассистента (Roo Code).

## Быстрый старт

1. `др` — сводка всех проектов
2. `проект 76` — выбрать проект 76
3. `стз 1` — детализация задачи 1

## Команды

| Команда | Действие |
|---------|----------|
| `др` / `сводка` / `итого` | Сводка всех проектов |
| `проект N` / `прN` | Выбрать проект |
| `стз N` | Детали задачи N |
| `заводим задачу` | Создать задачу |
| `обнови статус N.M` | Изменить статус этапа |
| `закрываем задачу N` | Завершить задачу |
| `откладываем задачу N` | Отложить задачу |
| `отк отчет` | Открыть последний отчёт |
| `правила дорожной карты` | Показать правила |

## Структура

```
ДорожнаяКарта/
├── projects.json        ← Единый список проектов
├── ПравилаДорожнойКарты.md
├── README.md
├── ДорожнаяКарта.md     ← Единая сводка
├── 1-76/                ← Проект 76
│   ├── tasks.json       ← Задачи (источник истины)
│   ├── ДорожнаяКарта.md ← Детализация
│   └── отчеты/          ← HTML-отчёты
```
