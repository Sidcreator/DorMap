## Подробно

### 1. Категория команд «📄 Отчёты»

| Фраза | Что делает |
|-------|-----------|
| «отк отчет» / «отк от» / «отк отчет текущий» / «отк от тек» | Открывает последний созданный HTML-отчёт в браузере |
| «отк отчет ИмяФайла» | Открывает отчёт по имени файла (например: «отк отчет Отчет_4.1_Анализ») |
| «отк отчет N.N» | Открывает отчёт, привязанный к этапу N.N (колонка **Отчёт** в таблице этапов) |

**Формат имени файла отчёта:**
```
Отчет_N.N_КраткоеОписание.html
```
Пример: `Отчет_4.1_Анализ_v1_analytics_stocks.html`

---

### 2. Колонка «Отчёт» в таблицах этапов

В таблицы этапов (уровень 2) добавлена колонка `Отчёт` между `Время (мин)` и `Журнал`:

```
| № | Этап | Описание | Статус | Дата | Время (мин) | Отчёт | Журнал |
```

Если к этапу привязан HTML-отчёт — ссылка `[Отчёт](отчеты/Отчет_N.N_...html)`. Если нет — `—`.

---

### 3. Синхронизация правил между проектами

При любом изменении правил (новая команда, колонка, формат) изменения вносятся во **все 4 файла**:

| № | Файл | Назначение |
|---|------|-----------|
| 1 | `1-76/ДорожнаяКарта.md` | Проект 1 |
| 2 | `2-76_in_КОРП/ДорожнаяКарта.md` | Проект 2 |
| 3 | `3-КОРП/ДорожнаяКарта.md` | Проект 3 |
| 4 | `ДорожнаяКарта.md` | Единая сводка |

**Процесс:** изменить единые правила → обновить протокол и таблицы в каждом проекте → обновить журналы → обновить единую сводку.

---

### 4. Команда «синхронизация»

```
синхронизация
```

AI проверяет состояние всех проектов, выявляет расхождения и применяет актуальные правила.
Использовать: после ручного редактирования, после обновления `ПравилаДорожнойКарты.md`, при подозрении на рассинхронизацию.

---

### 5. Два файла при обновлении

При каждом значительном изменении системы создаются **два файла** (в `ДорожнаяКарта/`):

| Файл | Содержит |
|------|----------|
| `План_СистемаДорожныхКарт_v(N+1).md` | Полный план с обновлёнными шаблонами |
| `Новое в ДорожнойКарте_v(N+1).md` | Краткое описание изменений версии |

---

### 6. Структура папок (v12+)

```
ДорожнаяКарта/
├── План_СистемаДорожныхКарт_v12.md    ← план

├── ПравилаДорожнойКарты.md            ← единые правила
├── ДорожнаяКарта.md                   ← единая сводка
├── 1-76/
│   ├── ДорожнаяКарта.md
│   ├── ЖурналИзменений.md
│   ├── отчеты/                        ← HTML-отчёты

│   ├── ...
│   └── отчеты/
└── 3-КОРП/
    ├── ...
    └── отчеты/
```

Планы хранятся в `ДорожнаяКарта/` (не в `1сexpert/планы/` как раньше).
