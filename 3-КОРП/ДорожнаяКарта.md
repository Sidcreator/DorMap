## Конвенции

### Конвенция: Лимит активных задач

- **Максимум 5 активных задач** (не отложенных) в дорожной карте
- При создании 6-й задачи без явного указания приоритета → **автоматически отложить** (перенести в секцию 7)
- Причина автооткладывания: «Превышен лимит активных задач (максимум 5)»
- Если одна из 5 задач закрыта (✅) → можно вытащить задачу из отложенных

### Конвенция: Перенос задач в активные

- При переносе задачи из выполненных или отложенных в активные задаче присваивается **новый номер активной задачи**.
- Новый номер выбирается по порядку среди активных задач: если активных задач нет, номер будет `1`; если активные задачи есть, используется следующий свободный номер.
- Старый номер фиксируется в описании задачи и журнале изменений как исторический источник переноса.
- Этапы, связанные планы и команды доступа перенумеровываются под новый активный номер; старый номер можно оставить алиасом, если по нему уже были ссылки.

### Конвенция: Учёт времени в минутах

- **Единица измерения:** минуты (целое число)
- **Колонка «Время (мин)»** — фактическое время, затраченное на задачу/этап
- **Правила заполнения:**
  - При старте задачи/этапа: `—` (прочерк)
  - При завершении: вписать количество минут
  - При промежуточном статусе: `~N мин` (приблизительная оценка)
- **Итог по времени** — секция в конце дорожной карты

## Протокол управления (ключевые фразы)

| Фраза | Действие |
|-------|----------|
| «заводим задачу в дорожную карту» | Создать задачу → дор карта 1 |
| «дор карта 1» / «др» / «др1» / «итого» / «сводка» / «дорожная карта» | Показать **все проекты** из [`projects.json`](../projects.json) с их разделами: активные + выполненные + отложенные + итого |
| «дор карта 2» / «др2» | Активные задачи текущего проекта + детализация |
| «статус задачи N» | Детализация задачи N в текущем выбранном проекте |
| «обнови статус N.M» | Изменить статус этапа |
| «закрываем задачу N» | Пометить готово + запросить время (мин) |
| «откладываем задачу N» | Отложить |
| «верни задачу N в активные» | Перенести из выполненных/отложенных в активные с новым номером |
| «итого время» | Сводка времени (мин / ч:мин) |
| «затрачено N мин на задачу M» | Вписать время |
| «правила дорожной карты» | Показать/сформировать правила |
| «обновить правила дорожной карты» | Зафиксировать изменения |
| «синхронизация» | Синхронизировать правила и протокол между всеми проектами |
| «отк отчет» / «отк от» / «отк отчет текущий» / «отк от тек» | Открыть последний HTML-отчёт |
| «отк отчет ИмяФайла» | Открыть отчёт по имени из `отчеты/` |
| «отк отчет N.N» | Открыть отчёт, привязанный к этапу N.N (по колонке Отчёт) |
